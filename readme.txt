De javascript checkt of er tags zijn met de zelfde value als de naam van een tabel met de class seg. Als hij die gevonden heeft zorgt hij ervoor dat als de checkbox gecheckt is de tabel zichtbaar is! Bij het toevoegen van een nieuwe promotiemogelijkheid moet je er dus voor zorgen dat de attribute "name" van de tabel en de value van de checkbox hetzelfde zijn. Vergeet ook niet de juiste functie als onClick toe te voegen bij de checkbox.
